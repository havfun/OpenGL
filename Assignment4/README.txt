Instructions:
------------

To compile this assignment:
$ make

This will produce executable file render_2d" inside bin folder.

--------------------------------------------------------------------------

$ cd bin
$ ./assignment2 ./data/bunny.ply

Key Codes :
	l : Enable/Disable Lighting
	c : Color set to default/Normals
	a : Draw Wireframe
	s : Draw Splats
	d : Draw Polygons
	m : Bigger Splats
	n : Smaller Splats
	+ : Zoom In
	- : Zoom Out
	q : Quit
	
Rotation:
	Mouse Right Click: Set Rotation Axis
	Left/Right Keys: Rotate Model.
