Instructions:
------------

To compile this assignment:
$ make

This will produce executable file render_2d" inside bin folder.

--------------------------------------------------------------------------

$ cd bin
$ ./render_2d

Key Codes :
	R	:Reset
	1	:Wine Glass
	2	:Butterfly
	3	:Crane
	r	:Rotate Right
	l	:Rotate Left
	+	:Zoom In
	-	:Zoom Out
  Curve Type	:
	B	:Bezier
	L	:Lagrange
    
  Esc/Q/q	: Quit Program
